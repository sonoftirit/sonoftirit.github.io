# mustafatiritoglu.github.io

Personal portfolio website built with HTML, CSS, and JavaScript — no frameworks, no build tools, just clean code.

## Live Site

[mustafatiritoglu.github.io](https://mustafatiritoglu.github.io)

## About

This is my personal portfolio site showcasing my projects, skills, and background as a Computer Science student at the University of British Columbia (UBC). Designed and built from scratch.

## Features

- Responsive layout — works on desktop, tablet, and mobile
- Smooth scroll fade-in animations using the Intersection Observer API
- Mobile hamburger navigation menu
- Project cards with tech stack tags and GitHub links
- Skills section covering languages, frameworks, and developer tools
- Contact section with email, GitHub, and LinkedIn

## Built With

- HTML5
- CSS3 (custom properties, flexbox, grid, media queries)
- Vanilla JavaScript (no libraries or frameworks)
- Google Fonts — Poppins & Inter
